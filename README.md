# AtliQ Hardwares – Data Analytics & Business Insights Project

## 🚀 Project Overview

This project was completed as part of the Codebasics Resume Project Challenge, where I worked as a Data Analyst for AtliQ Hardwares — a simulated yet realistic business environment.
The objective was to transform raw data into actionable insights and help the management team make faster, data-driven decisions.

## 🔍 Business Problem
 ◆ Lack of quick, data-driven decision-making  
 ◆ Difficulty in extracting meaningful insights from large datasets  
 ◆ Gap between raw data and strategic business actions  

## 📊 Analytics & Insights

  **Project Objective:**
  
  ✔ Extract and transform raw data using SQL  
  ✔ Generate ad-hoc insights for business decision-making  
  ✔ Identify key trends across sales, finance, and operations  
  ✔ Build interactive dashboards for stakeholders  
  ✔ Present insights in a clear and actionable format  

  **Purpose of Data Analytics:**  
  Enable management to make informed decisions by converting complex datasets into simple, actionable insights.
  
  **Importance of Data Analysis:**  
  Helps uncover hidden patterns, track performance metrics, improve operational efficiency, and support strategic planning.
  
  **Role of Insights:**  
  Drive business growth, optimize processes, identify opportunities, and reduce decision-making time.

## 📈 Dashboard & Visualization

Project Objective:

  ✔ Build interactive dashboards using Power BI  
  ✔ Visualize KPIs and business metrics effectively  
  ✔ Enable dynamic filtering for better exploration  
  ✔ Simplify complex data for non-technical stakeholders  

**Purpose of Dashboards:**  
Provide a visual representation of business performance for quick understanding and decision-making.

**Importance of Visualization:**  
Improves clarity, enhances storytelling, and helps stakeholders grasp insights faster.

**Role of Dashboards:**  
Monitor performance, identify trends, and support strategic discussions.

 ➤ Data modeling and transformation using Power Query  
 ➤ Creating calculated measures using DAX  
 ➤ Building interactive dashboards in Power BI  
 ➤ Data cleaning and preprocessing

 ➤ Created a structured PowerPoint presentation for stakeholders
